## 条目格式

每条条目是一个 **script** —— AI 遇到特定情境时的行动规则，附带预期后效（用于验证规则是否成立）。

| 字段 | 说明 |
|---|---|
| **强度** | ★–★★★★★，星级即稳定度 |
| **触发情境** | 什么情形下这条规则被激活 |
| **行动模式** | AI 应执行的动作 |
| **预期后效** | 执行此行动后预期用户/世界如何反应（用于验证规则有效性） |
| **发现日期** | 首次观察日期 |
| **证据** | 支撑观察的具体事件 |
| **related** | 关联条目 |
