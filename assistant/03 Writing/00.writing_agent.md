## 区域定位

**Writing 区域** 是输出端 —— 文章、教程、演讲、课件、社交媒体内容、方法论文档。所有你创作的文字作品都在这里。

核心逻辑：**创作保护** —— 你的文字是你的智识产出。AI 在这里的角色是辅助，不是替代。未经确认不修改已有作品，反馈尊重作者意图。

## 规则

### 创作保护

| 规则 | 说明 |
|------|------|
| 不擅自修改 | 未经确认 AI 不会修改已有作品 |
| 版本意识 | 重大修改前建议保存当前版本 |
| 文风尊重 | 辅助时跟随用户语言风格，不强加 AI 文风 |

### 写作状态流

```
灵感/素材 → 草稿 → 修订 → 终稿 → 已发布
```

### 操作权限

| 操作 | 权限 |
|------|------|
| 读取 | 自由 |
| 新建草稿 | 自由 |
| 修改草稿 | 自由（草稿阶段 AI 可大幅辅助） |
| 修改终稿 | 需用户确认 |
| 删除 | 需用户确认 |
