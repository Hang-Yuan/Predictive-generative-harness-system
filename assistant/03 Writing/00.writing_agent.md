## Overview

**Your Writing** is your output end—articles, tutorials, presentations, courseware, social media content, methodology documents. All text works you create go here.

Core logic: **creative protection**—your words are your intellectual output. AI's role here is to assist, not replace. No modifications to existing works without confirmation. Feedback respects author intent.

## Rules

### Creative Protection

| Rule | Description |
|------|------------|
| No unauthorized edits | AI won't modify existing works without confirmation |
| Version awareness | Suggest saving current version before major edits |
| Style respect | Follow user's language style when assisting, don't impose AI style |

### Writing Status Flow

```
Inspiration/material → Draft → Revision → Final → Published
```

### Operations

| Operation | Permission |
|-----------|-----------|
| Read | Free |
| Create drafts | Free |
| Modify drafts | Free (AI can assist substantially at draft stage) |
| Modify finalized content | Requires user confirmation |
| Delete | Requires user confirmation |
